# git init

Initialize a new Git repository in the current directory.

## Usage

```bash
git init
```

## Description

This command creates a new `.git` subdirectory in your current working directory. This subdirectory contains all the necessary Git metadata for the new repository.

## Example

```bash
mkdir my-project
cd my-project
git init
```

After running `git init`, you'll see the message:
> Initialized empty Git repository in /path/to/my-project/.git/
